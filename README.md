# README

## Tech

- Typescript
- Express
- MongoDB
- Docker

## Run the server

### script command

**Start**

```
yanr start
```

**Build**

```
yanr buile
```

### docker-compose

**Start**

```
docker-compose up -d
```

**Stop**

```
docker-compose down
```

### Run with script (recommend)

Recommend run the script to start the server.

```
bash ./scripts/restart-docker-compose.sh
```

### Receipt schema columns explanation

| schema columns | receipt data                           |
| -------------- | -------------------------------------- |
| store          | fist line                              |
| tel            | Tel                                    |
| gstRge         | GST Reg.                               |
| date           | Date, Time                             |
| serialNo       | Receipt ID                             |
| details        | Between `+-----+` line and empty line  |
| payment        | Before subTotal                        |
| subTotal       | subTotal                               |
| tender         | TENDER                                 |
| change         | CHANGE                                 |
| qty            | QTY                                    |
| total          | Total                                  |
| inclusive      | INCLUSIVE                              |
| gst            | GST                                    |
| tag            | Not in receipt data. Filled in by user |
