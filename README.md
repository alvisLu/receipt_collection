## How to run

**Start**

```
yanr start
```

**Build**

```
yanr buile
```

### docker-compose

**Start**

```
docker-compose up -d
```

**Stop**

```
docker-compose down
```

### Run with script

```
bash ./scripts/restart-docker-compose.sh
```
